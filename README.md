# Szereg Talora dla funkcji sinusa

Projekt zawiera zadanie z Algorytmów Numerycznych. Celem zadania było obliczenie funkcji sinusa przy pomocy Szeregu Taylora na cztery różne sposoby:
- V1 - sumowanie elementów szeregu obliczanych bezpośrednio ze wzoru Taylora w kolejności od początku
- V2 - sumowanie elementów szeregu obliczanych bezpośrednio ze wzoru Taylora w kolejności od tyłu
- V3 - sumowanie elementów szeregu od początku, ale obliczając kolejny wyraz szeregu na podstawie poprzedniego 
- V4 - obliczając kolejny wyraz szeregu na podstawie poprzedniego (tak jak w wersji V3), ale sumując od końca jak w V2

Otrzymane wyniki porównane zostały z wynikami bibliotecznej funkcji sinusa
